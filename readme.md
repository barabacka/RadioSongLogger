### RadioSongLogger

A simple code to save online radio playlist to file.
One or more stations...
to one or dedicated text files.

----
for use needs to add `config.json` file to the same folder:

```config.json
{
	"radio list": [
		[
			"https://radio url 1.mp3",
			"file to save 1.txt"
		],
		[
			"https://radio url 2.mp3",
			"file to save 2.txt"
		],
	]
}
```
